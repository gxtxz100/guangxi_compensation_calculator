# 广西人身损害赔偿计算器

这是一个针对广西壮族自治区人身损害赔偿项目的自动计算Python程序，具有友好的交互式界面，能够自动计算各项赔偿金额并生成Word文档。

## 功能特点

- ✅ 交互式图形界面，操作简单直观
- ✅ 自动计算各项赔偿项目
- ✅ 支持城镇/农村户籍类型
- ✅ 生成格式规范的Word文档
- ✅ 基于广西最新赔偿标准

## 赔偿项目

程序支持计算以下赔偿项目：

1. **医疗费** - 实际发生的医疗费用
2. **住院伙食补助费** - 按住院天数计算（默认100元/天）
3. **营养费** - 根据医嘱或实际需要
4. **交通费** - 实际发生的交通费用
5. **住宿费** - 实际发生的住宿费用
6. **误工费** - 根据月收入和误工天数计算
7. **护理费** - 根据护理天数和护理人数计算（默认150元/天）
8. **残疾赔偿金** - 根据伤残等级和户籍类型计算
9. **残疾辅助器具费** - 实际发生的费用
10. **被扶养人生活费** - 根据被扶养人年龄和数量计算
11. **死亡赔偿金** - 如适用，按户籍类型计算
12. **丧葬费** - 如适用，按标准计算
13. **精神损害抚慰金** - 根据实际情况

## 使用方法

### 运行程序
```bash
python guangxi_compensation_calculator.py
```

### 操作步骤

1. **填写基本信息**
   - 输入受害人姓名、年龄
   - 选择户籍类型（城镇/农村）
   - 输入事故发生日期

2. **填写各项费用**
   - 根据实际情况填写各项费用
   - 未填写的项目将按0计算
   - 部分项目有默认值（如住院伙食补助费、护理费）

3. **选择伤残等级**（如适用）
   - 从下拉菜单中选择伤残等级（1-10级）
   - 如无伤残，可不选择

4. **填写被扶养人信息**（如适用）
   - 输入被扶养人数量
   - 输入被扶养人年龄（用逗号分隔，如：5,10,15）

5. **计算赔偿**
   - 点击"计算赔偿"按钮
   - 查看计算结果

6. **导出Word文档**
   - 点击"导出Word文档"按钮
   - 选择保存位置
   - 生成格式规范的Word文档

## 注意事项

1. **数据准确性**：本程序使用的赔偿标准为示例数据，实际使用时请更新为广西最新官方标准。

2. **计算结果**：本计算结果仅供参考，实际赔偿金额以法院判决为准。

3. **证明材料**：各项费用需提供相应的票据和证明材料。

4. **专业咨询**：如对计算结果有疑问，请咨询专业律师。

## 赔偿标准说明

程序中的赔偿标准基于《中华人民共和国民法典》及相关司法解释，参考《2025年广西壮族自治区道路交通事故人身损害赔偿项目计算标准》。

主要标准包括：
- 城镇居民人均可支配收入：45,259元/年
- 农村居民人均纯收入：19,455元/年
- 城镇居民人均消费支出：29,500元/年
- 农村居民人均消费支出：15,400元/年
- 住院伙食补助费：100元/天
- 护理费标准：150元/天
- 丧葬费：50,000元

**重要提示**：请在使用前确认并更新为最新的官方标准数据。
